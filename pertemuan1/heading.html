HEADING, PARAGRAF, LINK, LIST

  • Heading (Judul)

    • Elemen h1, h2, h3, h4, h5, dan h6 dalam HTML digunakan untuk menandai judul-judul di halaman web. h1 merupakan judul paling penting dengan tingkat kepentingan tertinggi, sedangkan h6 adalah judul dengan tingkat kepentingan terendah

  • Paragraph (Paragraf)

    • Elemen p digunakan untuk menandai paragraf teks di halaman web. Paragraf adalah blok teks yang dibatasi oleh baris kosong atau elemen paragraf lainnya.

  • Link (Tautan)

    • Elemen a digunakan untuk membuat tautan atau hyperlink ke halaman web lain, dokumen, atau sumber daya. Atribut href digunakan untuk menentukan alamat URL tautan.

  • List (Daftar)
    • Ada dua jenis daftar dalam HTML: daftar terurut (ordered list) dan daftar tidak terurut (unordered list).

    • Ordered List (Daftar Terurut): Elemen ol digunakan untuk membuat daftar yang memiliki nomor atau angka sebagai penomoran. Elemen li digunakan untuk menandai setiap item dalam daftar.
    • Unordered List (Daftar Tidak Terurut): Elemen ul digunakan untuk membuat daftar yang menggunakan simbol atau ikon sebagai penanda. Elemen li digunakan untuk menandai setiap item dalam daftar.
