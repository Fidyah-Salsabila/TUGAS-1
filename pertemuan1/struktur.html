STRUKTUR HTML

Struktur utama di HTML terdapat tag HEAD, TITLE, BODY, HTML, FOOTER, dll

  1. Tag HTML

    2. Elemen HTML adalah elemen akar yang mengelilingi semua elemen HTML di halaman web. Semua elemen HTML berada di dalam tag HTML.

  2. Tag HEAD

    3. Elemen HEAD berisi informasi tentang halaman web yang tidak akan ditampilkan secara langsung kepada pengguna. Ini termasuk elemen seperti Meta, Title, dan tautan ke berkas eksternal.

  3. Tag BODY

    4. Elemen BODY berisi semua konten yang akan ditampilkan di halaman web. Ini termasuk teks, gambar, tautan, paragraf, judul, dan elemen-elemen lainnya yang akan dilihat oleh pengguna.

  4. Tag TITLE

    5. Elemen TITLE digunakan untuk menentukan judul halaman web yang akan ditampilkan di tab peramban.

  5. Tag FOOTER

    6. Elemen FOOTER berisi informasi yang terletak di bagian bawah halaman web, seperti hak cipta, tautan ke halaman lain, atau catatan kaki.
